在Ubuntu上安裝並設定WordPress
 8 / 23, 2013  Linux , Ubuntu , 教學
現在人們常常想要有一個在網路上能發表自己聲音的地方。除了像Facebook、Twitter等等，你是否想要有一個自己的Blog。WordPress 就是你的最佳選擇。趕快進來看看如何在Ubuntu上安裝WordPress！

什麼是WordPress?
根據WordPress中文網：
“WordPress 是一個開放原始碼網誌/內容管理系統，專注於美學、網頁標準和易用性的個人發佈平台。WordPress 兼具自由與無價，你可以下載並免費使用。”
簡單來講：WordPress 是一個簡單易用的界面，用來發佈個人的網站，Blog等。
本站也是使用WordPress 架構而成
想知道更詳細的解說：友站Noob’s Space曾經寫過一篇WordPress的詳細解說。
安裝WordPress
安裝Apache Server
Apache Server是一個開放源碼的網頁伺服器，是安裝WordPress前必須要安裝的軟體。
在Ubuntu中，按Ctrl-Alt-T開啟終端機。
輸入
sudo apt-get install apache2

apt-get是Ubuntu中通常是跟套件有關係的指令，而“apt-get install 套件名稱” 就是安裝套件的意思。
輸入使用者密碼。

是否繼續進行。輸入“Y”

安裝MySQL Server
MySQL是開放源碼的資料庫管理系統，也是安裝WordPress前的必須。
一樣在終端機中輸入：
sudo apt-get install mysql-server

安裝到一半會要求輸入root的密碼，很重要，請不要忘記。

再次確認密碼。

安裝WordPress
在終端機輸入：
sudo apt-get install wordpress
等待安裝完成。注意！WordPress 還沒安裝完成，請繼續下一個步驟。
設定MySQL
注意！！此小節的每一個步驟在輸入完都要案Enter(Return)鍵。
在終端機輸入：
sudo mysql -u root -p
MySQL會要求輸入密碼，只要輸入之前設定的root密碼即可。

輸入：
create database yourdatabase;
create database是MySQL指令創建資料庫(顧名思義create=創立，database=資料庫)
yourdatabase請替換成你想為WordPress 設定的資料庫名稱。
注意分號不能少喔！

接下來輸入：
create user yourusername@localhost;
create user就是創建使用者喔！youusername替換為你要的database帳號。

繼續輸入
set password for yourusername@localhost= password(‘password’);
這段指令是在設定yourusername這個使用者的密碼。
yourusername是你先前設定的帳號，’password’ 內的password請替換為你想為這個user設定的密碼。

再來輸入：
grant all privileges on yourdatabase.* to yourusername@localhost identified by ‘password’;
這會給予yourusername這個使用者所有yourdatabase的權限。
youdatabase是你先前創立的database名稱，yourusername是先前的帳號，password是先前設定的密碼。

最後輸入：
flush privileges;
這會載入我們之前更改的權限內容。
之後再輸入：
exit

安裝WordPress資料
到WordPress正體中文網首頁下載WordPress 。

到下載資料夾中解壓縮zip檔。

接下來，開啟終端機並輸入：
sudo cp -r /home/igene/下載/wordpress  /var/www
cp是copy的指令，後面要多打-r是因為我們要複製的是整個資料夾。
/var/www是Apache Server預設的網頁位置。

接下來我們要製作一份WordPress的設定檔。
終端機中輸入：
sudo cp /var/www/wordpress/wp-config-sample.php  /var/www/wordpress/wp-config.php
這將會複製範例檔並建立一個設定檔。

接下來我們要更改設定檔，讓WordPress 使用我們先前創立的database。
sudo gedit /var/www/wordpress/wp-config.php
gedit是Ubuntu Desktop內建的文字編輯軟體，這個指令是要電腦以文字編輯器開啟wp-config.php這個檔案。
這將會開啟文字編輯器。

找到這一段：

更改database_name_here為先前設定的database名稱。
更改username_here為先前設定的帳號名稱。
更改password_here為先前設定的使用者密碼。
