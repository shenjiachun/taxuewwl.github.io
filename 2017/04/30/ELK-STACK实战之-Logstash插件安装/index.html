ELK-STACK实战之--Logstash插件安装

By SnoWalker

发表于 2017-04-30

文章目录
  1. 1. 1、下载插件包
  2. 2. 2、安装
  3. 3. 3、测试

官网提及5.x版本的Logstash，去掉了17个插件。所以，如果需要使用，我们只能手动安装了。

开始：

1、下载插件包

假设我需要下载exec包，那么在github网站 https://github.com/logstash-plugins/logstash-output-exec 下载 exec包。
logstash-output-exec-master.zip

2、安装

将exec包上传到Logstash所在的服务器并解压：

/app/soft/logstash-output-exec-master

修改Logstash的Gemfile文件，再最后添加一行：

gem "logstash-output-exec", :path => "/app/soft/logstash-output-exec-master"

最后执行安装：

bin/logstash-plugin install --no-verify

返回
Installing…
Installation successful
OK……
